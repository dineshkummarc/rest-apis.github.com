Una rápida introducción a CURL

$ which curl
  $ man curl
  $ curl google.com
  $ curl -v google.com
  $ curl -I google.com
  $ curl -i google.com
  $ curl -v httpbin.org/get?hello=world
  $ curl -v --data param1=data1 httpbin.org/post

¿Qué es HTTP?

  • Usa el ciclo solicitud-respuesta
  • Enteramente basado en archivos de texto con poco formato “especial”
